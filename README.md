# GuideCap
GuideCap, görme, işitme, konuşma ve denge bozukluğu olan bireyler için geliştirilmiş akıllı ve giyilebilir bir destek şapkasıdır. Sistem; Arduino tabanlı sensörler, yapay zeka destekli veri analizi, gerçek zamanlı nesne algılama ve mobil uygulama entegrasyonu ile çevresel farkındalık ve güvenliği artırmayı hedefler.
